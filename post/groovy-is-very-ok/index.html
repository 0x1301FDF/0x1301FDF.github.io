闭包火葬场。
  • each { } : return ==> (Java) continue ----终止当前闭包执行，不影响后面的循环
  • find { } : return true ==> (Java) break ----结束所有闭包，返回当前it；
    return false ==> (Java) contine ----终止当前闭包执行，不影响后面的循环，如果没有return true，最终结果是null，find用于查找值，闭包参数用作终止条件（故需要布尔类型），也可以骚操作用来迭代。

     name = "jack"
     age = 12
     //不return自己，就会返回age，最后jack会变成12
